metro ui

Overview

A modern recreation of the iconic Windows Phone Metro UI, bringing the beautiful and intuitive tile-based interface to the web. Experience the smooth animations, gesture-based navigation, and minimalist design that made Windows Phone unique.

Features

  • Gesture Navigation

    Swipe right to open info panel, swipe left for search

  • Live Tiles

    Dynamic tiles with live updating content and animations

  • Customizable Accent Colors

    Personalize your experience with custom theme colors

  • Custom Backgrounds

    Set your own background images with blur effects

  • Control Center

    Quick access to settings and toggles from any screen

  • Universal Search

    Search for apps from anywhere in the interface

  • Music Player

    Xbox Music-inspired player with now playing screen

  • Weather App

    Real-time weather information with OpenWeatherMap API

  • Camera Integration

    Access device camera with mirrored preview

  • iOS Web App Support

    Install to home screen for a native app experience

Technology

Built with HTML5, CSS3, and vanilla JavaScript. Features include:

  • CSS3 animations and transforms
  • Touch gesture detection
  • Local storage for settings persistence
  • Canvas API for background effects
  • MediaDevices API for camera access
  • Fetch API for weather data
